# SabaleSanchita03-CodeAlpha_Image_Gallery
Interactive Image Gallery built using HTML, Tailwind CSS, and JavaScript with upload, favorites, delete, and lightbox features.
# 🖼️ Interactive Image Gallery

This is a responsive Image Gallery web application built using:

- HTML
- Tailwind CSS
- JavaScript

## 🚀 Features
- Image filtering (Nature, Travel, Animals)
- Lightbox view
- Next/Previous navigation
- Upload your own images
- Delete images
- Favorites system ❤️
- Dark mode 🌙
- Keyboard controls

## 📸 Live Demo
(https://sabalesanchita03.github.io/SabaleSanchita03-CodeAlpha_Image_Gallery/)

## 📂 Project Structure
- index.html
- script.js
- image.css
- images/

## 🎯 Purpose
This project was built as part of frontend development internship practice to improve UI/UX and JavaScript skills.
